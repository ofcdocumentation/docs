Projects are the core unit of collaboration on the platform. This guide explains how to discover, join, and actively contribute to projects.

## What Are Projects?

Projects are collaborative workspaces where teams come together around a shared goal. Each project has its own members, files, tasks, and communication channels.

## Finding Projects

### Browse the Project Directory

Navigate to **Platform Areas → Projects** to browse all available projects. You can filter by:

- Category or topic
- Organization
- Status (active, completed, seeking contributors)
- Date created

### Search

Use the global search bar at the top of the platform to search for projects by name or keyword.

## Joining a Project

Once you’ve found a project you’re interested in:

1. Click on the project to view its profile
2. Review the project’s description, goals, and current members
3. Click **Request to Join** or **Join** (depending on the project’s settings)
4. For private projects, your request will be sent to the project admin for approval

<Note>
  Some projects are open to anyone, while others require an invitation or approval from the project owner.
</Note>

## Contributing to a Project

Once you’re a member, you can:

- **View and upload files** in the project’s file library
- **Participate in discussions** on the project’s activity feed
- **Access project resources** shared by other members
- **Track project progress** using the built-in tools

## Creating a New Project

Have an idea you want to bring to life? Create your own project:

1. Click **\+ New Project** from the Projects directory
2. Fill in your project’s name and description
3. Set visibility (public or private)
4. Invite collaborators
5. Click **Create Project**
