The platform’s file management system lets you store, organize, and share files across your projects, groups, and organizational profile.

## Supported File Types

The platform supports a wide range of file types:

- **Documents** – PDF, Word (.docx), Excel (.xlsx), PowerPoint (.pptx)
- **Images** – JPG, PNG, GIF, SVG, WebP
- **Video** – MP4, MOV, AVI (up to 2GB)
- **Audio** – MP3, WAV, AAC
- **Data** – CSV, JSON, XML
- **Archives** – ZIP, TAR.GZ

## Uploading Files

### Uploading to a Project

1. Navigate to your project
2. Click the **Files** tab
3. Click **Upload Files** or drag and drop files into the upload area
4. Add a title and description (optional but recommended)
5. Set visibility (defaults to the project’s visibility setting)
6. Click **Upload**

### Uploading to a Group

Same process as uploading to a project, but from your group’s **Files** tab.

### Uploading as a Resource

To publish a file as a standalone resource visible beyond your project or group, use [Resource Publishing](/platform-areas/resource-publishing/about).

## Organizing Files

### Folders

Create folders to keep your files organized:

1. In the **Files** tab, click **New Folder**
2. Name the folder
3. Drag files into the folder or upload directly to it

### File Metadata

Add metadata to make files easier to find:

- **Title** – A human-readable name
- **Description** – What the file contains or how to use it
- **Tags** – Keywords for search and filtering
- **Date** – When the content was created or is relevant to

## Sharing Files

### Sharing Within the Platform

You can share files with specific people or make them available to all project/group members by adjusting visibility settings.

### Sharing via Link

Public files have a shareable link that anyone can use to view or download the file, even without an account.

1. Click on the file
2. Click **Share**
3. Copy the link

## File Limits

| Plan | Storage Per User | Max File Size |
| --- | --- | --- |
| Free | 5 GB | 100 MB |
| Pro | 50 GB | 1 GB |
| Organization | 500 GB | 2 GB |

<Note>
  Storage is shared across all your projects and groups. Contact your administrator if you need additional storage.
</Note>

## Managing Existing Files

### Edit File Details

1. Click on the file to open its detail view
2. Click the **Edit** button (pencil icon)
3. Update the title, description, or tags
4. Click **Save**

### Replace a File

To upload a new version while keeping the same link and metadata:

1. Open the file detail view
2. Click **Replace File**
3. Upload the new version

### Delete a File

1. Open the file detail view
2. Click **⋯** → **Delete**
3. Confirm deletion

<Warning>
  Deleting a file is permanent. Any links pointing to that file will break.
</Warning>
