Map components let you display your work geographically — showing where you operate, where your beneficiaries are located, and how impact is distributed across regions.

## Map Types

### Point Map

Plots individual locations as dots on a map.

**Use cases:** Showing program locations, partner organization locations, event locations.

### Choropleth Map

Colors regions (countries, states, counties) based on data values. Darker or more saturated colors indicate higher values.

**Use cases:** Showing reach by region, comparing outcomes by state, visualizing funding distribution.

### Heat Map

Shows the density or concentration of activity across a geographic area.

**Use cases:** Where beneficiaries are concentrated, identifying underserved areas.

### Bubble Map

Like a point map, but the size of each dot represents a data value.

**Use cases:** Showing volume of services by location, comparing office sizes.

## Setting Up a Map Component

### Adding Locations

You can add location data in two ways:

**Manual Entry**

1. Click **\+ Add Location**
2. Enter an address or search for a location
3. Add any associated data values
4. Repeat for each location

**Import from File**

1. Prepare a CSV with columns for address (or latitude/longitude) and any data values
2. Click **Import Locations**
3. Upload your file and map the columns

### Configuring Display

- **Base map style** – Choose street map, satellite, or minimal
- **Zoom level** – Set the default zoom when the map loads
- **Bounding region** – Constrain the map to a specific geographic area
- **Color scheme** – Choose colors for your data ranges

## Using Platform Location Data

If your projects or groups have location data, you can pull it directly into your map:

1. In the map configuration, select **Platform Data** as the source
2. Choose the projects or groups to include
3. Select which location fields to display

## Interactivity

Map components are interactive by default. Viewers can:

- Pan and zoom the map
- Click on points or regions to see detail popups
- Toggle between layers (if you’ve set up multiple data layers)

You can configure what information appears in popups from the **Popup Settings** panel.
