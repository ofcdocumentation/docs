Find answers to common questions about Impact Dashboards.

## General

<AccordionGroup>
  <Accordion title="Who can create an Impact Dashboard?">
    Any platform member can create a personal Impact Dashboard. To create a dashboard on behalf of an organization, group, or project, you’ll need Editor access or higher.
  </Accordion>

  <Accordion title="How many dashboards can I create?">
    Free accounts can create up to 3 dashboards. Pro and Organization plans have unlimited dashboards. Contact your administrator to check your plan limits.
  </Accordion>

  <Accordion title="Can I make my dashboard private?">
    Yes. When creating or editing a dashboard, set the visibility to **Private**. Only you (and organization admins) will be able to see it.
  </Accordion>

  <Accordion title="Can multiple people edit the same dashboard?">
    Yes. You can share edit access with specific people or all members of a project/group. Go to **Dashboard Settings → Collaborators** to manage access.
  </Accordion>
</AccordionGroup>

## Data & Integration

<AccordionGroup>
  <Accordion title="How often does the data update?">
    Platform data (from your projects and groups) updates in real time. Imported data (from CSV files) is static unless you re-import. Connected integrations update on a schedule you configure.
  </Accordion>

  <Accordion title="What file formats can I import?">
    You can import data from CSV (.csv) and Excel (.xlsx) files. The maximum file size for imports is 10 MB.
  </Accordion>

  <Accordion title="Can I connect to external data sources?">
    Yes, for Pro and Organization plan users. Supported integrations include Google Sheets, Airtable, and common database connectors. See the Integrations documentation for a full list.
  </Accordion>

  <Accordion title="Can I export my dashboard data?">
    Yes. From any dashboard, click **Export** to download the underlying data as a CSV, or export the entire dashboard as a PDF. Individual components can also be exported as image files (PNG).
  </Accordion>
</AccordionGroup>

## Display & Sharing

<AccordionGroup>
  <Accordion title="Can I embed a dashboard on my website?">
    Yes. Click **Share → Embed** to get an embed code for the entire dashboard or individual components. Embedded dashboards update automatically as your data changes.
  </Accordion>

  <Accordion title="Can I share a dashboard with people who don't have an account?">
    Public dashboards can be viewed by anyone with the link, including people without accounts. Platform Members and Connected Only dashboards require an account.
  </Accordion>

  <Accordion title="How do I share a dashboard with my team?">
    Click **Share** and enter the email addresses or usernames of the people you want to share with. You can choose to give them view-only or edit access.
  </Accordion>
</AccordionGroup>

## Troubleshooting

<AccordionGroup>
  <Accordion title="My data isn't showing up. What should I check?">
    1. Make sure your data source is properly connected in the component settings.
    2. Check that your date range filters aren’t excluding your data.
    3. If using imported data, verify the column mapping is correct.
    4. Try refreshing the dashboard by clicking the refresh icon.
  </Accordion>

  <Accordion title="The dashboard is loading slowly. How can I speed it up?">
    Dashboards with many components or large datasets may load slowly. Try:

    - Reducing the number of components on a single dashboard
    - Using date range filters to limit data volume
    - Breaking large dashboards into multiple focused dashboards
  </Accordion>
</AccordionGroup>
