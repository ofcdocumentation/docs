Modules, particularly the **Chart** and **Measurable Results** modules, require data sources. Users can connect data in the following ways:

- **Live Data Sources**: Pull real-time data from API integrations, project field data, or web documents stored in the system.
- **Historical Data Sources**: Upload CSV files, paste spreadsheet data, or manually enter historical data.

# Connecting Chart Modules with Data

There are a few ways to connect data to modules to be able to display data

## Connect to Project Data

Charts can connect to existing project data in the platform. You can source data from multiple projects. 

- Choose  ‘Fetch from Project List’ from the Data Source Method Menu and select an existing project.
- Then, select from the templates to select the form you would like to connect.
- Last, select specific fields you would like to appear in your chart, through the field titled “data keys.”

## Connect to a Spreadsheet 

This option works for spreadsheets that you have created using the platform, in your files or under the attachment modal. 

Choose ‘Fetch from Spreadsheet List’ from the Data Source Method Menu and choose from previously uploaded Spreadsheets.

## **Connect to API**

Choose ‘Fetch from Remote API’ from the Data Source Method Menu and enter the Remote URL. Please reach out to us directly at [<u>support@openfuturecoalition.org</u>](mailto:support@openfuturecoalition.org) if you are interested in exploring this option, and we can help you get it set up.

## **Connecting Modules to Historical Data (Manual Entry Spreadsheet and CSV)**

**To add data to a spreadsheet manually:** Select ‘Manual Spreadsheet’ from the Data Source Method Menu and Enter data directly into the spreadsheet.

**To paste from CSV:** Select ‘Manual Data (Comma-separated)’ from the Data Source Method and paste in data.
