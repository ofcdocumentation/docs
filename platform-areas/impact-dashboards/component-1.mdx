The Dashboard Publishing Tool is built around **modules**, which are individual content blocks that users can add and configure. Each module serves a specific purpose, whether it’s displaying data, linking resources, or adding visuals.

# Creating a Layout 

\[Layout screenshot\] 

To ensure your page is responsive and nicely spaced, we recommend dragging in a new container (with single or multiple rows) before you add each row of content. Available formats include single-column, two-column, three-column, and 3/7 split.

# Adding Modules 

Once you’ve added a row, simply drag a module from the left-hand panel into the canvas. It will then walk you through upload or configuration options with a popup window.

**Types of Modules**

- **\[attachment icon\] Attachment**: Upload or embed files, including videos from YouTube/Vimeo.
- **\[icon\] Image Carousel**: Display multiple images from uploaded or existing files.
- **\[icon\]Measurable Results**: Integrate data from measurable results reporting forms.
- **\[icon\] Card**: Create structured information cards with titles, descriptions, and preview images.
- **\[icon\] Chart**: Visualize data through various chart types (e.g., bar, pie, scatter, radar).
