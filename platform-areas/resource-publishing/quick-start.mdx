The resource publishing tool is designed to support shared learning, coordination, and thoughtful curation of information across the Open Future Coalition community as well as for public audiences.

Resources published through the tool may include:

- Practical tools, templates, and guides
- Project examples and implementation playbooks
- Training materials, frameworks, or research
- Curated links to external knowledge or supporting documentation

Think of a resource as a **container for knowledge**, not just a file. Publishing a resource provides context, attribution, and intentional sharing—so work can travel responsibly. Organizations can choose whether resources are publicly accessible, visible only to selected members, or shared within a specific Group context.

Public resources are added to The Open Impact Resource Library, a curated, shared collection of practices, tools, and knowledge intended to support the wellbeing of the planet, people, and place. \[link to info page about the resource library itself)

\[Screenshot showing “My Resources" and “Add New”\]

1. Go to **My Resources → Add New**
2. Add **Basic Info** (title, images, tags, sponsor)
3. Set **Description \+ Visibility & Privacy**
4. Attach **Files & Links**
5. **Save** to  continue editing later or **Submit** – prepares the resource for publishing (note: once you click “submit,” the resource publishing workflow will need to be re-opened in order to add or modify files), , then **Publish files** via the Publishing Wizard \[link\]

| **Submit ≠ Publish** <ul><li>**Submit** makes the resource card visible where shared</li><li>**Publish** controls whether attached files are accessible</li></ul> |
