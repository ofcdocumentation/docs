This guide covers all the ways you can add content to the platform’s resource directory.

## Content Types

### File Uploads

Upload files directly to the platform:

**Supported formats:**

- Documents: PDF, Word (.docx), Excel (.xlsx), PowerPoint (.pptx)
- Data: CSV, JSON, XML
- Images: PNG, JPG, SVG
- Archives: ZIP

**Upload process:**

1. Click **\+ Publish Resource → Upload a File**
2. Select or drag your file
3. Complete the metadata form
4. Set visibility and publish

### External Links

Link to content hosted elsewhere (your website, Google Drive, GitHub, etc.):

1. Click **\+ Publish Resource → Add a Link**
2. Paste the URL
3. The platform will attempt to auto-populate metadata from the linked page
4. Review and complete the metadata form
5. Set visibility and publish

<Note>
  External links point to content on other sites. You’re responsible for keeping the link current if the destination moves.
</Note>

### Written Content

Create long-form written content directly on the platform using the built-in editor:

1. Click **\+ Publish Resource → Write Content**
2. Use the rich text editor to write and format your content
3. Add images, tables, and callouts as needed
4. Complete the metadata form
5. Publish

**Rich text editor features:**

- Headings and formatting
- Bulleted and numbered lists
- Tables
- Images and file embeds
- Hyperlinks
- Callout blocks (info, warning, tip)

## Managing Published Resources

### Editing a Resource

1. Navigate to the resource
2. Click **Edit Resource**
3. Make your changes
4. Click **Save** (changes are live immediately)

### Publishing a New Version

When you upload a new version of a file:

1. Open the resource
2. Click **Upload New Version**
3. Add version notes describing what changed
4. Previous versions are archived and accessible via **Version History**

### Unpublishing a Resource

To temporarily take a resource offline:

1. Open the resource
2. Click **Settings → Unpublish**
3. The resource is hidden from the directory but not deleted

### Deleting a Resource

1. Open the resource
2. Click **Settings → Delete**
3. Confirm deletion

<Warning>
  Deleting a resource is permanent. All analytics data and comments are also deleted. Consider unpublishing instead.
</Warning>
