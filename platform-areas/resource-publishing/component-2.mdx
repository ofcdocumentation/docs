Great metadata is the difference between a resource that sits unnoticed and one that gets discovered and used widely.

## Why Metadata Matters

The platform’s search algorithm and directory filters rely heavily on metadata to surface relevant content. Well-tagged resources show up in more searches, get recommended to the right people, and ultimately have more impact.

## Metadata Fields

### Required Fields

| Field | Description | Tips |
| --- | --- | --- |
| **Title** | The name of the resource | Clear, specific, searchable |
| **Description (short)** | 1-2 sentences shown in search results | Front-load the most important info |
| **Resource Type** | Category of resource | See types below |
| **Publication Date** | When the content was created | Use the original publication date, not upload date |

### Recommended Fields

| Field | Description | Tips |
| --- | --- | --- |
| **Full Description** | Detailed overview | Include context, methodology, and how to use it |
| **Tags** | Freeform keywords | Use specific terms your audience would search for |
| **Focus Areas** | Predefined topic categories | Select all that apply |
| **Geographic Focus** | Where the content applies | Be specific where possible |
| **Language** | Primary language | Required for multi-language search |

### Attribution Fields

| Field | Description |
| --- | --- |
| **Author(s)** | Individual creators |
| **Publisher** | Organization responsible for the resource |
| **License** | How others may use the content |
| **Citation** | How to credit the resource |

## Resource Types

Choose the type that best describes your resource:

| Type | Use For |
| --- | --- |
| Report | Research findings, evaluations, assessments |
| Guide | How-to instructions, best practices, toolkits |
| Dataset | Raw or processed data files |
| Template | Reusable documents or frameworks |
| Policy | Laws, regulations, policy documents |
| Multimedia | Videos, podcasts, presentations |
| News & Updates | Timely news and announcements |
| Funding Opportunity | Grants, fellowships, contracts |

## Tagging Best Practices

### Use Specific Tags

❌ Too vague: `education`, `health`, `community` ✔ More specific: `early childhood education`, `mental health`, `community resilience`

### Include Audience Tags

Tag by who should use the resource: `for nonprofits`, `for researchers`, `for policymakers`, `for practitioners`

### Include Geographic Tags

Tag the geographic scope: `global`, `national`, `northeast US`, `urban`, `rural`

### Add Methodology Tags

For research: include methodology terms like `qualitative`, `randomized control trial`, `survey data`

## Licenses

Choose the right license for your content:

| License | What It Allows |
| --- | --- |
| **All Rights Reserved** | No reuse without permission |
| **CC BY** | Free to use with attribution |
| **CC BY-SA** | Free to use; derivatives must use same license |
| **CC BY-NC** | Free for non-commercial use only |
| **CC0** | No rights reserved; public domain |
