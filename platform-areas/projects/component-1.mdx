Effective member management ensures the right people have the right access to your project.

## Project Roles

| Role | Capabilities |
| --- | --- |
| **Creator/Admin** | The person who creates the project is the primary administrator. Currently, only the creator can edit the project description and manage overall membership settings. |
| **Project Managers** | Designated managers can create and assign tasks, manage discussions, and send calendar invites. |
|  |  |
|  |  |
|  |  |

### Adding a Project Manager 

To add managers to a project, navigate to the project page, click on the gear icon in the banner, and then ‘Edit Project’. Add members from the Managers drop down. If someone you would like to assign as a manager is not appearing in this list, make sure they are in the managers list of a sponsoring Organization (which can be found under the Organization's profile settings.)

## Inviting Members

### Send an Invitation

1. Navigate to **Project → Members**
2. Click **\+ Invite Members**
3. Enter an email address or search for a platform user by name
4. Choose a role
5. Add an optional personal note
6. Click **Send Invitation**

Invited users will receive an email notification and an in-app alert.

### Invite via Link

For open projects, you can share a join link:

1. Go to **Project → Settings → Membership**
2. Enable **Join by Link**
3. Copy and share the link

<Note>
  Join links can be disabled at any time from Project Settings.
</Note>

## Managing Existing Members

### Changing a Member’s Role

1. Go to **Project → Members**
2. Click the role dropdown next to the member’s name
3. Select the new role

### Removing a Member

1. Go to **Project → Members**
2. Click **⋯** next to the member’s name
3. Select **Remove from Project**

### Blocking a Member

If a member is disruptive, you can remove and block them:

1. Follow the steps to remove a member
2. Check the **Block from rejoining** option
3. Confirm

## Membership Requests

For closed projects, people can request to join. To manage requests:

1. Go to **Project → Members → Pending Requests**
2. Review each request (includes the requester’s profile and any message)
3. Click **Approve** or **Decline**

## Member Notifications

Admins can send announcements to all project members:

1. Go to **Project → Members**
2. Click **Send Announcement**
3. Write your message
4. Choose notification channels (in-app, email, or both)
5. Click **Send**
