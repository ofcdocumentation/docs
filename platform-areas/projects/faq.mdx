Answers to common questions about using Projects on the platform.

## Getting Started

<AccordionGroup>
  <Accordion title="Who can create a project?">
    Any platform member can create a project. If you want to create a project on behalf of an organization, you’ll need Editor access or higher within that organization.
  </Accordion>

  <Accordion title="Is there a limit to how many projects I can create?">
    Free accounts can be active in up to 10 projects. Pro accounts have no limit. Organization plans may have different limits — check with your administrator.
  </Accordion>

  <Accordion title="Can I transfer ownership of a project?">
    Yes. Go to **Project → Settings → Transfer Ownership**, search for the new owner, and confirm. The new owner must be a current project member.
  </Accordion>
</AccordionGroup>

## Membership & Access

<AccordionGroup>
  <Accordion title="How do I make a project private?">
    Go to **Project → Settings → Visibility** and select **Private**. Only current members will be able to see the project.
  </Accordion>

  <Accordion title="Can people join without an invitation?">
    For Open projects, yes. For Closed projects, they can request to join and an admin must approve. For Private projects, they can only join if directly invited.
  </Accordion>

  <Accordion title="Can I have guests (people who can view but not edit)?">
    Yes. Assign the **Viewer** role to anyone who should have read-only access.
  </Accordion>

  <Accordion title="What happens to content when a member leaves?">
    Content created by a member remains in the project when they leave. The member’s name will still appear on files and posts they created.
  </Accordion>
</AccordionGroup>

## Files & Content

<AccordionGroup>
  <Accordion title="Can I move files between projects?">
    You can download files from one project and upload them to another. Direct moves between projects aren’t currently supported.
  </Accordion>

  <Accordion title="What happens to files when a project is archived?">
    Files are preserved when a project is archived. Members can still view and download files, but new uploads are disabled.
  </Accordion>

  <Accordion title="Is there a file size limit?">
    File size limits depend on your plan. See [File Management](/platform-fundamentals/file-management) for details.
  </Accordion>
</AccordionGroup>

## Projects and Organizations

<AccordionGroup>
  <Accordion title="Can a project be affiliated with multiple organizations?">
    Yes. A project can be affiliated with multiple organizations or groups. Go to **Project → Settings → Affiliations** to add or remove affiliations.
  </Accordion>

  <Accordion title="Can an organization control a project's settings?">
    Organization admins can manage projects affiliated with their organization, even if they’re not project members themselves.
  </Accordion>
</AccordionGroup>
